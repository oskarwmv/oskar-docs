# oskar_uber

Documentation page reserved for `oskar_uber`.

## Current Notes

- Uber-style ride request app for FiveM.
- Driver/passenger ride state handled by Lua client and server files.
- Bot ride profiles and destinations are configured in `config.lua`.
- Pickup and destination coordinates should stay on streets, sidewalks, or safe curb-side points.
